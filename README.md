## Homebrew tap for [fir](https://github.com/kfet/fir)

This cask is maintained automatically by GoReleaser on each release.

## macOS (via Homebrew)

The tap is private — you need to be granted access first.

```bash
brew tap kfet/fir git@github.com:kfet/homebrew-fir.git
brew install --cask fir
```

To upgrade:

```bash
brew upgrade --cask fir
```

## Linux

Download the latest binary from the [releases page](https://github.com/kfet/fir/releases/latest), or use the install script:

```bash
curl -fsSL https://raw.githubusercontent.com/kfet/fir/main/install.sh | bash
```

Or use `fir update` to self-update an existing installation.
